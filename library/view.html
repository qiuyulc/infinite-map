视图控制（铺满/居中/锁定）#

本页介绍在“预览/只读”场景下，如何控制相机（camera）让某个节点铺满屏幕，以及如何锁定视图拖动。

1）核心概念：camera#

Infinite Map 通过相机描述当前视图：

1type Camera = { x: number; y: number; zoom: number }
  • (x, y)：当前视图左上角在 世界坐标系 中的位置
  • zoom：缩放倍率（越大越“放大”）

你可以通过 apiRef 读取/设置：

1const cam = apiRef.current?.getCamera()
2apiRef.current?.setCamera({ x: 0, y: 0, zoom: 1 }, { immediate: true })

2）让某个节点“铺满屏幕”（fit to node）#

思路：把节点的 world rect 按 viewport（像素）计算一个合适的 zoom，并把节点中心对齐到屏幕中心。

2.1 计算公式（推荐）#

1type Rect = { x: number; y: number; w: number; h: number }
2type Viewport = { w: number; h: number }
3type Camera = { x: number; y: number; zoom: number }
4
5export function fitRectToViewport(rect: Rect, vp: Viewport, paddingPx = 24): Camera {
6  const vw = Math.max(1, vp.w - paddingPx * 2);
7  const vh = Math.max(1, vp.h - paddingPx * 2);
8
9  const zoom = Math.min(vw / rect.w, vh / rect.h);
10
11  const cx = rect.x + rect.w / 2;
12  const cy = rect.y + rect.h / 2;
13
14  return {
15    zoom,
16    x: cx - vp.w / (2 * zoom),
17    y: cy - vp.h / (2 * zoom),
18  };
19}

2.2 在 React 中调用（示例）#

1import { useLayoutEffect, useRef } from 'react';
2import type { InfiniteMapApi, NodeData } from '@qiuyulc/infinite-map';
3
4export function Preview({ nodes }: { nodes: NodeData[] }) {
5  const apiRef = useRef<InfiniteMapApi | null>(null);
6  const containerRef = useRef<HTMLDivElement | null>(null);
7
8  useLayoutEffect(() => {
9    const api = apiRef.current;
10    const el = containerRef.current;
11    if (!api || !el) return;
12
13    const node = nodes.find((n) => n.id === 'cover');
14    if (!node) return;
15
16    const vp = { w: el.clientWidth, h: el.clientHeight };
17    const cam = fitRectToViewport({ x: node.x, y: node.y, w: node.width, h: node.height }, vp, 24);
18    api.setCamera(cam, { immediate: true });
19  }, [nodes]);
20
21  return (
22    <div ref={containerRef} style={{ width: '100vw', height: '100vh' }}>
23      <InfiniteMap nodes={nodes} apiRef={apiRef} editMode="readonly" />
24    </div>
25  );
26}

建议把调用时机放在：

  • 页面进入预览时
  • 容器尺寸变化时（ResizeObserver / window resize）
  • 节点尺寸变化时（例如图片加载后节点尺寸变化）

3）锁定视图拖动（禁止平移）#

如果你希望预览时不允许用户拖动画布：

1<InfiniteMap nodes={nodes} editMode="readonly" panEnabled={false} />

panEnabled=false 会禁用：

  • 空白拖拽平移
  • Space 平移模式
  • 触控板两指平移（wheel pan）

4）固定画布模式（海报 / 封面）#

默认相机以世界原点 (0,0) 为视口中心。如果你需要固定尺寸画布（例如 800×600 的海报），且希望节点坐标按"距左上角距离"来写：

1<InfiniteMap
2  nodes={nodes}
3  panEnabled={false}
4  zoomEnabled={false}
5  initialCamera={{ x: 0, y: 0, zoom: 1 }}
6/>
  • initialCamera={{ x: 0, y: 0, zoom: 1 }} 把视口左上角对齐世界原点
  • 节点 x: 30 即距画布左边 30px，无需手动换算
  • 搭配 panEnabled={false} 和 zoomEnabled={false} 锁定视口，模拟传统设计工具的固定画布体验
