编辑器定制#

本文介绍如何定制 Infinite Map 编辑器的 UI 和行为——工具栏、右键菜单、快捷键、主题、HUD 组件。

目录#

  1. 定制工具栏

  3. 定制快捷键

  5. 定制 HUD 组件
  6. 定制节点渲染

1. 定制工具栏#

按 key 快速排列#

不需要导入任何类型，传字符串数组即可（'|' = 分隔线）：

1const plugins = createDefaultEditorPluginsWithUI({
2  toolbar: {
3    enabled: true,
4    items: [
5      "history.undo",
6      "history.redo",
7      "|",
8      "view.zoomIn",
9      "view.zoomOut",
10      "|",
11      "edit.delete",
12    ],
13  },
14});

内置 key：history.undo history.redo view.zoomOut view.zoomIn view.resetZoom view.fitView view.centerView edit.delete

混合自定义项#

字符串和 ToolbarItem 对象可以混用：

1import type { ToolbarItem } from "@qiuyulc/infinite-map-editor";
2
3const plugins = createDefaultEditorPluginsWithUI({
4  toolbar: {
5    enabled: true,
6    items: [
7      "history.undo",
8      "history.redo",
9      "|",
10      { type: "command", id: "my.export", label: "导出", icon: <ExportIcon /> },
11      "|",
12      "edit.delete",
13    ],
14  },
15});

完整自定义#

1import type { ToolbarItem } from "@qiuyulc/infinite-map-editor";
2
3const customItems: ToolbarItem[] = [
4  // 分隔线
5  { type: "separator" },
6
7  // 运行命令的按钮
8  {
9    id: "undo",
10    type: "command",
11    commandId: "history.undo",
12    label: "撤销",
13    shortcut: "⌘Z",
14  },
15  {
16    id: "redo",
17    type: "command",
18    commandId: "history.redo",
19    label: "重做",
20    shortcut: "⇧⌘Z",
21  },
22
23  // 分隔线
24  { type: "separator" },
25
26  // 自定义渲染按钮
27  {
28    id: "custom",
29    type: "custom",
30    label: "我的按钮",
31    render: (ctx) => {
32      return <button onClick={() => console.log("clicked")}>自定义</button>;
33    },
34  },
35];
36
37const plugins = createDefaultEditorPluginsWithUI({
38  toolbar: {
39    enabled: true,
40    items: customItems,
41    position: "top-left", // 或 'top-right'
42  },
43});

ToolbarItem 类型#

1type ToolbarItem =
2  | { type: "separator" }
3  | {
4      type: "command";
5      id: string;
6      commandId: string;
7      label: string;
8      shortcut?: string;
9      disabled?: boolean;
10    }
11  | {
12      type: "custom";
13      id: string;
14      label: string;
15      render: (ctx: MapContext) => React.ReactNode;
16      disabled?: boolean;
17    };

默认工具栏包含：undo / redo / zoom in / zoom out / fit view / center view / delete。

2. 定制右键菜单#

按 key 快速排列#

1const plugins = createDefaultEditorPluginsWithUI({
2  contextMenu: {
3    enabled: true,
4    items: ["edit.copy", "edit.paste", "edit.duplicate", "|", "edit.delete"],
5  },
6});

内置 key：edit.copy edit.cut edit.paste edit.duplicate edit.delete z.bringToFront z.bringForward z.sendBackward z.sendToBack edit.group edit.ungroup edit.lock edit.unlock edit.hide edit.showAll view.fitView view.centerView view.fitSelection view.centerSelection

ContextMenuItem 类型#

1type ContextMenuItem =
2  | { type: "divider" }
3  | {
4      type: "command";
5      id: string;
6      commandId: string;
7      label: string;
8      shortcut?: string;
9      disabled?: boolean;
10      visible?: (selectedIds: string[]) => boolean;
11    }
12  | {
13      type: "submenu";
14      id: string;
15      label: string;
16      visible?: (selectedIds: string[]) => boolean;
17      children: ContextMenuItem[];
18    }
19  | {
20      type: "custom";
21      id: string;
22      label: string;
23      visible?: (selectedIds: string[]) => boolean;
24      render: (ctx: MapContext, selectedIds: string[]) => React.ReactNode;
25    };

3. 定制快捷键#

通过 shortcuts.commandShortcuts 覆盖或禁用默认快捷键：

1const plugins = createDefaultEditorPluginsWithUI({
2  shortcuts: {
3    commandShortcuts: {
4      // 覆盖
5      "history.undo": "Mod+Shift+Backspace",
6      "edit.delete": "Mod+Backspace",
7
8      // 禁用一个快捷键（设为 null）
9      "edit.cut": null,
10
11      // 添加额外快捷键（多键绑定）
12      "edit.copy": "Mod+C, Ctrl+Insert",
13    },
14  },
15});

快捷键语法：Mod+Key 格式（Mod 在 Mac 上为 Cmd，Windows 上为 Ctrl）。支持用逗号分隔多个快捷键绑定同一个命令。

所有内置命令 ID 见 命令速查表。

4. 外部插件贡献 UI#

你的自定义插件可以向工具栏和右键菜单贡献按钮/菜单项，无需手动配置 items 数组。

贡献工具栏按钮#

1import { STORE_KEYS } from "@qiuyulc/infinite-map";
2import type { ToolbarItem } from "@qiuyulc/infinite-map-editor";
3
4const myPlugin: InfiniteMapPlugin = {
5  id: "my-plugin",
6  setup: (ctx) => {
7    // 注册到 toolbar items registry
8    const prev = ctx.store.get<ToolbarItem[]>(STORE_KEYS.toolbarItems) ?? [];
9    ctx.store.set(STORE_KEYS.toolbarItems, [
10      ...prev,
11      {
12        id: "my-action",
13        type: "command",
14        commandId: "my-plugin.action",
15        label: "我的操作",
16      },
17    ]);
18  },
19};

Toolbar 插件会自动合并 STORE_KEYS.toolbarItems 到工具栏中。

贡献右键菜单项#

1import { STORE_KEYS } from "@qiuyulc/infinite-map";
2import type { ContextMenuItem } from "@qiuyulc/infinite-map-editor";
3
4const myPlugin: InfiniteMapPlugin = {
5  id: "my-plugin",
6  setup: (ctx) => {
7    const prev =
8      ctx.store.get<ContextMenuItem[]>(STORE_KEYS.contextMenuItems) ?? [];
9    ctx.store.set(STORE_KEYS.contextMenuItems, [
10      ...prev,
11      {
12        id: "my-menu-item",
13        type: "command",
14        commandId: "my-plugin.action",
15        label: "我的操作",
16      },
17    ]);
18  },
19};

5. 定制 HUD 组件#

自定义标尺#

1const plugins = createDefaultEditorPluginsWithUI({
2  rulers: {
3    enabled: true,
4    thickness: 32, // 标尺宽度/高度（px）
5  },
6});

自定义小地图#

1const plugins = createDefaultEditorPluginsWithUI({
2  minimap: {
3    enabled: true,
4    width: 300,
5    height: 200,
6    includeOrigin: true, // 是否强制包含原点 (0,0)
7    showStats: false, // 是否显示调试统计
8  },
9});

自定义缩放条#

1const plugins = createDefaultEditorPluginsWithUI({
2  zoomDock: {
3    enabled: true,
4    snapToggleEnabled: true, // 是否显示吸附开关
5  },
6});

完全自定义 HUD 布局#

如果你需要超出默认配置的 HUD 定制（例如把小地图放在左上角），可以不用默认组装，手动控制每个 HUD 插件：

1import { composePlugins } from "@qiuyulc/infinite-map-editor";
2import { createDefaultEditorPlugins } from "@qiuyulc/infinite-map-editor";
3import {
4  createMinimapPlugin,
5  createRulersPlugin,
6  createZoomDockPlugin,
7  createToolbarPlugin,
8} from "@qiuyulc/infinite-map-editor";
9
10const core = createDefaultEditorPlugins({
11  /* core options */
12});
13
14const plugins = composePlugins([
15  ...core,
16  createMinimapPlugin({ width: 300, height: 200 }),
17  createRulersPlugin({ thickness: 24 }),
18  createZoomDockPlugin({ snapToggleEnabled: true }),
19  // 工具栏放在最后，确保在最上层
20  createToolbarPlugin({ enabled: true, position: "top-right" }),
21]);

6. 定制节点渲染#

通过 InfiniteMap 的 renderNode / renderNodeContent props 自定义节点外观：

1<InfiniteMap
2  nodes={nodes}
3  plugins={plugins}
4  renderNode={({ node, children, style }) => (
5    <div style={style}>{children}</div>
6  )}
7  renderNodeContent={({ node }) => (
8    <div style={{ padding: 12 }}>
9      <strong>{node.label ?? node.id}</strong>
10      <div style={{ fontSize: 12, color: "#888" }}>
11        ({node.x.toFixed(0)}, {node.y.toFixed(0)})
12      </div>
13    </div>
14  )}
15/>
  • renderNode：控制节点的包裹容器（可以加自定义背景、边框等）
  • renderNodeContent：控制节点内部内容

详见 组件 API。

7. 定制主题#

Infinite Map 提供了亮色和暗色两套主题，支持深度定制：

1import { InfiniteMap } from "@qiuyulc/infinite-map";
2import {
3  lightTheme,
4  darkTheme,
5  mergeTheme,
6  themeToCSSVars,
7} from "@qiuyulc/infinite-map-editor";
8
9// 基于亮色主题创建自定义主题
10const myTheme = mergeTheme(lightTheme, {
11  node: {
12    backgroundColor: "#f0f4ff",
13    borderColor: "#4a90d9",
14    borderRadius: "8px",
15  },
16  selection: {
17    borderColor: "#ff6b35",
18    handleColor: "#ff6b35",
19  },
20  grid: {
21    color: "#e8ecf0",
22  },
23  snapGuide: {
24    color: "#ff6b35",
25  },
26});
27
28// 转换为 CSS 变量注入
29const cssVars = themeToCSSVars(myTheme);
30
31<div style={cssVars as React.CSSProperties}>
32  <InfiniteMap nodes={nodes} plugins={plugins} />
33</div>;

或者使用 InfiniteMapThemeProvider：

1import { InfiniteMapThemeProvider } from "@qiuyulc/infinite-map-editor";
2
3<InfiniteMapThemeProvider theme={myTheme}>
4  <InfiniteMap nodes={nodes} plugins={plugins} />
5</InfiniteMapThemeProvider>;

主题结构见 主题定制。
